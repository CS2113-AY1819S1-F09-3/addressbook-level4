Invités - Team F09-3

=== This repo contains the source code for Group F09-3's CS2113T Project

Invités is a Windows application that caters to organisers of different events (such as weddings, orientation camps, networking sessions, product launches, etc). It helps event planners reduce their workload while planning events by providing them a convenient platform to do all the planning.

Features:

General

  1. Side tab, where event planners will be able to see what events they are planning.

For each event

  1. Overarching detail fields

    1. Name of event

    2. Date and Time of event

    3. Venue of event

    4. Dress code for event

    5. Number of guests so far

  2. Table of details of guests

    1. Name of guest

    2. Mobile number of guest

    3. Email address of guest

    4. Dietary Requirement of guest

    5. Ticket number/QR code of guest
